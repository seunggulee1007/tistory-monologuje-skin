# Changelog

이 프로젝트의 주요 변경 사항을 기록합니다.
[Keep a Changelog](https://keepachangelog.com/ko/1.1.0/) 형식을 따르며, [Semantic Versioning](https://semver.org/lang/ko/)을 사용합니다.

## [1.0.0] - 2025-01-01

### Added

- **다크/라이트 모드**: OS 설정 자동 감지 + 수동 토글 + localStorage 저장, FOUC 방지
- **코드 하이라이팅**: highlight.js 기반 자동 언어 감지, 파일명 주석으로 언어 힌트
- **코드 복사 버튼**: 원클릭 코드 복사 + 언어 뱃지 표시
- **PlantUML 렌더링**: 코드블록의 PlantUML을 SVG 다이어그램으로 자동 변환
- **자동 목차(TOC)**: H2/H3 기반 목차 자동 생성 + 스크롤 스파이
- **반응형 디자인**: 1440/1280/1024/768/480px 5단계 브레이크포인트
- **키보드 단축키**: `/` 검색, `d` 다크모드, `t` 목차, `j/k` 이전/다음 글, `h` 홈, `?` 도움말
- **헤딩 앵커 링크**: 제목 호버시 링크 아이콘, 클릭시 섹션 URL 복사
- **읽기 위치 기억**: 이탈 후 복귀시 "이어 읽기" 배너 표시
- **읽기 진행률 바**: 상단 스크롤 진행도 색상 바
- **예상 읽기 시간**: 한국어/영어 혼합 텍스트 기반 자동 계산
- **이미지 라이트박스**: 본문 이미지 클릭시 풀스크린 확대
- **이미지 지연 로딩**: Intersection Observer 기반 레이지 로딩
- **검색 오버레이**: 키보드 기반 블로그 내 검색
- **사이드바**: 카테고리, 최근글, 태그 클라우드
- **11개 커스텀 변수**: 강조 색상, 폰트, 레이아웃, 코드 테마 등 관리자 패널에서 설정
